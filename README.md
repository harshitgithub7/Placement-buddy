# Placement Buddy

Placement Buddy is a platform designed to improve communication between the **Training & Placement Office (TPO)** and **students**, while enabling students to learn from **alumni and professors with industry experience**.

The system helps students track company opportunities, apply for roles, optimize resumes, and connect with mentors.

# Frontend Setup
The frontend is built using:

* React
* TypeScript
* Vite

### Step 1: Navigate to frontend folder
cd frontend

### Step 2: Install dependencies
npm install

### Step 3: Start development server
npm run dev
Open in browser:
http://localhost:5173
